# The Foodie Hub

A beginner-friendly multi-page recipe website built using **HTML and CSS**.
This project focuses on practicing website structure, page linking, layouts, media embedding, typography, and reusable styling while creating a fun cooking-themed website.

**Live Demo:**## Live Demo

(https://madhushreerasal.github.io/The-Foodie-Hub/)

## About The Project

**The Foodie Hub** is a simple recipe website designed for beginners who are learning how to cook.
The homepage contains multiple recipe cards with videos, and each recipe opens on its own dedicated page containing:

* Ingredients
* Step-by-step procedure
* Notes and serving tips
* Images and media

This project was mainly created to strengthen foundational HTML and CSS skills through building a complete multi-page website.

## Features

* Multi-page website structure
* Recipe cards with videos
* Internal page navigation
* Embedded media (images and videos)
* Custom fonts using Google Fonts
* Styled recipe layouts
* Beginner-friendly recipe instructions
* Responsive viewport setup

## Technologies Used

* HTML5
* CSS3
* Google Fonts

## What I Practiced

Through this project, I practiced:

* Structuring webpages using HTML
* Linking multiple pages together
* Using CSS for layouts and styling
* Working with images and videos
* Creating reusable design sections
* Adding favicons and meta tags
* Organizing content clearly
